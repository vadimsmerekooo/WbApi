# WbApi
c# console project for wb api Core 7

###Input parameters:
- link to card 
  > Example: (https://www.wildberries.by/product?card=9000213&option=30191991);
- quantity;
- Berrer token (required parameter for send post method to server);
- Cookie param: _wbauid;

#Greate! The project works for different regions from domain ranges!
